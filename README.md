
# React Movies with Forms

## Description
Exercise with React.js showing a SPA with dynamic components and a simple form using the “controlled components”.

## Scripts

In the project directory, you can run:
### $ `npm install`
### $ `npm start`

## Technology Stack
- React.js
- Node.js
- Html
- Css
- Surge.sh
